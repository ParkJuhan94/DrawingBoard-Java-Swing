# DrawingBoard_Java_Swing
- Java의 Swing으로 GUI 인터페이스 만들기 (그림판 만들기)
- Library와 응용 프로그램을 별도의 프로젝트로 분리
- container는 자기가 갖고있는 모든 component 들을 paint 시켜준다.
- 다양한 component 구현 및 시연

### 그림판의 기능:
- 선분 / 사각형 / 타원 / 그룹화
- 선색 / 채움색
- 이동 / 복사 / 삭제
- 파일에 저장 / 불러오기
